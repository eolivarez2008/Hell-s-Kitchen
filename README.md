# Hell's Kitchen

Arena Shooter 2D en vue de dessus développé avec Godot 4.

## Gameplay

- Survive aux vagues d'ennemis (des aliments) de plus en plus difficiles
- Tes armes tirent automatiquement sur l'ennemi le plus proche
- Ramasse les piments pour acheter des améliorations entre les vagues
- La partie se termine quand ta vie tombe à 0

## Contrôles

| Touche  | Action           |
| ------- | ---------------- |
| Flèches | Déplacer le Chef |

## Progression

- Chaque vague dure plus longtemps et spawn plus d'ennemis
- Les ennemis gagnent en PV et en vitesse à chaque vague
- Le meilleur score (vague maximale atteinte) est sauvegardé localement

## Boutique (entre les vagues)

| Upgrade          | Coût  |
| ---------------- | ----- |
| Vie Max +20      | 30 🌶 |
| Vitesse +20      | 25 🌶 |
| Dégâts G/D +5    | 20 🌶 |
| Cadence G/D +0.1 | 35 🌶 |

## Stack
